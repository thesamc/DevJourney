# DevJourney

A responsive web development blog featuring an interactive card-based layout with a neon cyberpunk theme. Built with vanilla HTML, CSS, and JavaScript.

## Features

- Interactive card-flip animations
- Neon cyberpunk design system
- Responsive layout using CSS Grid
- Custom scrollbar styling
- Typing animation effect

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Space Grotesk, JetBrains Mono)

## Live Demo

[devjourney.samchutter.com](https://devjourney.samchutter.com)

## Setup

1. Clone the repository:
```bash
git clone https://github.com/thesamc/devjourney.git
```

2. Open `index.html` in your browser
